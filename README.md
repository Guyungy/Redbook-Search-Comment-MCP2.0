# 小红书自动搜索评论工具

一款基于 Playwright 开发的小红书自动化工具，作为 MCP Server 可接入 Claude for Desktop 等 MCP 客户端，实现智能搜索、内容获取和评论发布功能。

## 项目结构

项目已重构为模块化架构，便于维护和扩展：

```
Redbook-Search-Comment-MCP2.0/
├── xiaohongshu_mcp.py      # 主服务器文件，定义MCP工具接口
├── config.py               # 配置管理和全局变量
├── browser_manager.py      # 浏览器管理和登录功能
├── search_engine.py        # 搜索功能模块
├── content_analyzer.py     # 内容分析模块
├── comment_manager.py      # 评论管理模块
├── requirements.txt        # 项目依赖
└── README.md              # 项目说明
```

### 模块说明
- **config.py**: 环境变量配置、数据目录设置、URL处理工具
- **browser_manager.py**: Playwright浏览器初始化、登录状态管理
- **search_engine.py**: 基础搜索、智能搜索、深度分析功能
- **content_analyzer.py**: 笔记内容提取和分析
- **comment_manager.py**: 评论获取和发布功能

## 主要功能

### 1. 用户登录
- 持久化登录，扫码登录后保存登录状态
- 自动检测登录状态，无需重复登录
- 支持重置登录状态，解决登录失效问题

### 2. 智能搜索与内容获取
- **基础搜索**：根据关键词搜索小红书笔记，可指定返回数量
- **智能搜索**：AI Agent式深度搜索，自动分析任务意图、生成多个搜索策略、执行多轮搜索并智能筛选结果
- **深度搜索分析**：最高级搜索模式，不仅搜索还会自动分析每个笔记内容，提取趋势和关键信息，生成深度分析报告
- **获取笔记内容**：提取笔记标题、作者、发布时间和正文内容
- **获取评论**：获取笔记的评论信息，包括评论者、内容和时间

### 3. 智能评论
- **内容分析**：自动分析笔记领域（美妆、穿搭、美食、旅行等8个领域）
- **AI评论生成**：结合 MCP 客户端的 AI 能力生成自然评论
- **多种评论类型**：
  - 引流型：引导用户关注或私聊
  - 点赞型：简单互动获取好感
  - 咨询型：以问题形式增加互动
  - 专业型：展示专业知识建立权威

## 安装方法

### 1. 环境准备
确保系统已安装 Python 3.8 或更高版本。

### 2. 克隆项目
```bash
git clone <项目地址>
cd Redbook-Search-Comment-MCP2.0
```

### 3. 创建虚拟环境
```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows
.\venv\Scripts\activate
# Mac/Linux
source venv/bin/activate
```

### 4. 安装依赖
```bash
pip install -r requirements.txt
playwright install
```

### 5. MCP 配置
在 Claude for Desktop 的配置文件中添加：

```json
{
    "mcpServers": {
        "xiaohongshu MCP": {
            "command": "C:\\Users\\4AM\\Documents\\GitHub\\Redbook-Search-Comment-MCP2.0\\venv\\Scripts\\python.exe",
            "args": [
                "C:\\Users\\4AM\\Documents\\GitHub\\Redbook-Search-Comment-MCP2.0\\xiaohongshu_mcp.py",
                "--stdio"
            ]
        }
    }
}
```

**注意**：请将路径替换为你的实际项目路径。

## 使用方法

### 1. 启动服务
配置好 MCP 后，在 Claude for Desktop 中连接服务器。

### 2. 基本操作流程

1. **用户登录**
```python
login()  # 扫码登录，登录状态会自动保存

# 如果遇到登录问题，可以重置登录状态
reset_login()  # 重置登录状态
# 然后重新登录
login()  # 重新扫码登录
```

2. **搜索功能（三种模式）**

**基础搜索**：
```python
search_notes("护肤", 10)  # 简单关键词搜索，返回10个结果
```

**智能搜索**（推荐）：
```python
smart_search_notes("我想找一些关于敏感肌护理的笔记，特别是针对换季时期的护肤建议", 10)
# AI会自动分析任务意图，生成多个搜索策略，执行多轮搜索并智能筛选
```

**深度搜索分析**（最全面）：
```python
deep_search_and_analyze("帮我分析最近流行的日系穿搭趋势", True, 5)
# 不仅搜索，还会分析每个笔记内容，生成趋势报告和行动建议
```

3. **获取笔记内容**
```python
get_note_content("https://www.xiaohongshu.com/explore/xxx")  # 获取指定笔记的详细内容
```

4. **获取笔记评论**
```python
get_note_comments("https://www.xiaohongshu.com/explore/xxx")  # 获取笔记的评论信息
```

5. **智能评论**
```python
post_smart_comment("https://www.xiaohongshu.com/explore/xxx", "引流")  # 生成并发布智能评论
```

### 3. 基本操作

#### 登录小红书
```
帮我登录小红书账号
```

#### 搜索笔记（三种模式）

**基础搜索**：
```
帮我搜索小红书笔记，关键词为：美食
帮我搜索小红书笔记，关键词为旅游，返回10条结果
```

**智能搜索**（推荐）：
```
帮我智能搜索：我想找一些关于敏感肌护理的笔记，特别是针对换季时期的护肤建议
帮我智能搜索关于日系穿搭的内容，返回15条结果
```

**深度搜索分析**（最全面）：
```
帮我深度搜索并分析最近流行的日系穿搭趋势
帮我深度分析护肤领域的最新趋势，包含详细分析报告
```

#### 获取笔记内容
```
帮我获取这个笔记的内容：https://www.xiaohongshu.com/explore/xxxx
```

#### 获取笔记评论
```
帮我获取这个笔记的评论：https://www.xiaohongshu.com/explore/xxxx
```

#### 发布智能评论

### 3. 评论类型说明

- **点赞**：简单互动获取好感，提升曝光率
- **咨询**：以问题形式增加互动，引发博主回复

## 智能搜索功能详解

### 🔍 三种搜索模式对比

| 功能 | 基础搜索 | 智能搜索 | 深度搜索分析 |
|------|----------|----------|-------------|
| **适用场景** | 简单关键词查找 | 复杂任务理解 | 趋势分析研究 |
| **搜索策略** | 单一关键词 | 多轮智能搜索 | 多轮+内容分析 |
| **结果质量** | 基础匹配 | 智能筛选排序 | 深度分析报告 |
| **执行时间** | 快速 | 中等 | 较长 |
| **信息深度** | 标题+链接 | 标题+链接+评分 | 完整内容+趋势 |

### 🤖 智能搜索特性

- **任务意图分析**：自动识别搜索领域和核心需求
- **多策略搜索**：生成多个搜索关键词组合
- **智能去重**：自动去除重复结果
- **结果评分**：基于相关性对结果进行评分排序
- **搜索报告**：提供详细的搜索过程和策略分析

### 🧠 深度分析功能

- **内容提取**：自动获取每个笔记的详细内容
- **趋势分析**：统计热门领域、关键词分布
- **质量评估**：评估内容质量和完整性
- **行动建议**：基于分析结果提供实用建议

```bash

```

## 注意事项

## 常见问题

**Q: 遇到登录失效怎么办？**
A: 如果遇到"需要登录"或其他登录相关错误，可以先调用`reset_login()`重置登录状态，然后重新调用`login()`进行登录。

**Q: 如何解决登录问题？**
A: 确保网络连接正常，使用小红书APP扫描二维码登录

**Q: 智能搜索没有结果怎么办？**
A: 尝试更详细的任务描述，或使用基础搜索功能

**Q: 深度分析功能很慢？**
A: 这是正常现象，因为需要分析每个笔记的详细内容，建议减少分析数量

**Q: 评论发布失败？**
A: 检查账号状态和评论内容是否符合平台规范
